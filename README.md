# FlowMatch

Search thousands of ready-to-use n8n automation workflows and find the right template for your process. FlowMatch matches, indexes, and curates workflows automatically.

## Features

- **5,000+ Verified Workflows**: Real, unique n8n automation templates.
- **AI Matching Engine**: Natural language matching to locate the best templates for your tools.
- **Node normalizer**: Clean display titles for n8n components.
- **Advanced Security Scans**: Automated security, complexity, and duplicate checking.

## Getting Started

1. Install dependencies:
   ```bash
   npm ci
   ```
2. Build for production:
   ```bash
   npm run build
   ```
3. Run locally in development:
   ```bash
   npm run dev
   ```
